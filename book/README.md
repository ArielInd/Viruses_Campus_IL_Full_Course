# ספר לימוד – קורס נגיפים וחיסונים

ספר זה נבנה מתוך תמלולי השיעורים (course_transcripts) והומר למבנה לימודי קריא עם יעדי למידה, סיכומים, ותרגול לבחינות.

## מבנה התיקיות

- `book/`: קבצי הספר הסופיים.
- `ops/`: תוצרי ביניים, דוחות, ולוגים של תהליך העיבוד.

## בנייה מחדש

הריצו `run_pipeline.py` כדי לעבד מחדש את התמלולים ולבנות את הספר.
