# Tech Stack: Course Transcript Downloader

## Core Technologies
- **Language:** Python 3.x
- **Web Interaction:** Playwright (for handling dynamic content and authentication)
- **CLI Framework:** Argparse (built-in)

## Dependency Management
- **Package Manager:** pip
- **Manifest:** `requirements.txt`

## Configuration & Environment
- **Environment Management:** python-dotenv (for loading credentials from a `.env` file)

## Development Tools
- **Linter/Formatter:** Ruff (recommended for fast and comprehensive Python linting)
