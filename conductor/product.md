# Product Guide: Course Transcript Downloader

## Overview
A specialized tool designed to assist students in downloading lecture transcripts from the Campus IL course website for offline study and reference.

## Target Audience
- **Students:** Individuals enrolled in the "Viruses" course on Campus IL who prefer studying from text-based materials.

## Core Features
- **Bulk Download:** Capability to fetch all available course transcripts in a single execution.
- **Metadata Retention:** Saved files will include lecture titles and chapter names to maintain context.
- **Plain Text Output:** Transcripts are saved as `.txt` files for maximum compatibility and simplicity.

## Technical Constraints & Requirements
- **Authentication:** The tool must handle user authentication to access restricted course content.
- **Campus IL Integration:** Specifically tailored to navigate and extract data from the Campus IL platform.
